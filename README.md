# Comandos-dotnet-CLI
Comandos para el uso de dotnet CLI


## Ventajas del uso del dotnet CLI 

tabajar en cualquier sistema operativo y desde 
cualquier editor de texto

## Requerimientos para el uso de la los comandos

    dotnet --version
